## Overview
NumPy has powerful built-in statistical functions that allow you to analyze data quickly.

## Common Functions
-   `np.mean()`: Average
-   `np.median()`: Median (middle value)
-   `np.std()`: Standard Deviation (spread)
-   `np.var()`: Variance
-   `np.min()` / `np.max()`: Minimum / Maximum values
-   `np.argmin()` / `np.argmax()`: Index of min / max values

## Axis Argument
Most of these functions accept an `axis` argument:
-   `axis=0`: Operate along columns (vertical)
-   `axis=1`: Operate along rows (horizontal)
