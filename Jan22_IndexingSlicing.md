## 1. Indexing
Indexing allows you to access individual elements of an array.
-   **1D Arrays:** Similar to Python lists (0-based index).
-   **2D Arrays:** Access using `[row, column]`.

## 2. Slicing
Slicing allows you to access a subarray.
-   Syntax: `array[start:stop:step]`
-   **2D Slicing:** `array[row_start:row_stop, col_start:col_stop]`

## 3. Boolean Indexing
Boolean indexing allows you to select elements based on a condition (e.g., all elements greater than 5).
